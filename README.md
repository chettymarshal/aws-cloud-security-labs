# AWS Cloud Security Labs

This repository contains hands-on AWS cloud security projects built as part of a structured learning path toward AWS Security Specialty certification.

Each project focuses on real-world security architecture, logging, IAM design, and infrastructure hardening.


## Projects

### 01 - Secure EC2 Infrastructure
Traditional AWS architecture with VPC, EC2, IAM roles, and logging enabled.
